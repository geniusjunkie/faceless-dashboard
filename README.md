# Faceless Channel Intelligence Dashboard

**WNNR. analyst** — A self-contained intelligence dashboard surfacing newly-launched, high-performing faceless YouTube channels across Vidrush and Sleep niches.

## Features

- 📹 **Vidrush section** — Long-form (18–40 min), monetized faceless channels
- 🌙 **Sleep section** — Sleep stories, meditation, calm narration channels
- 📊 **Overflow pool** — All other surfaced channels
- 🔀 **Niche-Bend Strategy cards** — AI-generated pivot ideas for Bend Pool channels
- 🎬 **Recent-video toggles** — Per-channel video preview with thumbnails
- ⬇️ **CSV export** — Download recent video data per channel
- 🔍 **Live filter & sort** — Min views, age, RPM, search, sort by any column

## Data

Research-based estimates derived from publicly available niche benchmarks (2025–2026). RPM figures sourced from industry reports. All revenue figures are directional estimates, not live data.

## Deploy

Deployed as a static site via [Render](https://render.com). No server required — all data is baked into the HTML.

## Update

To generate a new report, replace `index.html` with an updated dashboard file and push to `main`.
